Chapter 3 · Deterministic models

The SIR model

We split a closed population into three compartments and track how people flow from susceptible to infectious to recovered over time.

The model is governed by a transmission rate beta and a recovery rate gamma. Their ratio defines the basic reproduction number, the average number of secondary cases produced by one infectious person in a fully susceptible population.

★Key idea

When R0 > 1 the pathogen can invade; when R0 < 1 each chain of transmission shrinks and the outbreak fades.

Defining the flows

The system of ordinary differential equations moves probability mass between compartments at each time step:

dS/dt = −β·S·I / N    dI/dt = β·S·I / N − γ·I    dR/dt = γ·I
iNote

N is assumed constant, so births, deaths, and migration are ignored on the timescale of a single outbreak.

We solve this numerically in R with deSolve:

# SIR model in R sir <- function(t, state, p) { with(as.list(c(state, p)), { dS <- -beta * S * I / N dI <- beta * S * I / N - gamma * I dR <- gamma * I list(c(dS, dI, dR)) }) } R0 <- beta / gamma # reproduction number
✓Tip

Start with R0 and a mean infectious period, then back out beta. It is far easier to reason about than a raw contact rate.

!Warning

The deterministic model hides randomness. With few infectives, stochastic fade-out can end an outbreak even when R0 > 1.

!Important

Always check units. If time is in days, gamma must be a per-day rate, not a period.
